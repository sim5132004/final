인천 지하철 시계열 분석

인천 광역시 지하철역 하차 유동인구 기준, 유동인구 혼잡도를 시계열 예측 분석

1. 예측모델: Neuralprophet

작년 7월, 올해 6월까지 부평역 하차인원을 일별로 기록한 데이터프레임에서 랜덤하게 10%의 행을 선택해 샘플링하여 일부 데이터를 추출하여 검증용데이터프레임으로 만들고 원본 데이터프레임에서 제거해 원본데이터를 학습데이터로 만들었습니다.


2. 데이터 수집 및 전처리

먼저, 인천 지하철 하차 인원과 관련된 데이터를 수집합니다. 이 데이터에는 하차역, 날짜 정보가 포함됩니다. 데이터를 정리하고 필요한 형식으로 전처리합니다. 예를 들어, 날짜와 시간 정보를 처리하고, 역의 위치 정보를 확보합니다.


3. 딥러닝 모델 구성

딥러닝 모델을 구성합니다. 이 모델은 인천 지하철 하차 인원과 관련된 데이터를 입력으로 받아 혼잡도를 예측하는 작업을 수행합니다. 모델은 일반적으로 입력 레이어, 여러 은닉 레이어, 그리고 출력 레이어로 구성됩니다. 데이터에 따라 모델의 아키텍처가 다를 수 있습니다.


4. 학습 데이터 준비

원본 데이터를 학습과 검증 데이터로 나눕니다. 일반적으로 학습 데이터는 모델을 학습하는 데 사용되고, 검증 데이터는 모델의 성능을 평가하는 데 사용됩니다. 여기서 원본 데이터프레임에서 일부 행을 랜덤하게 선택하여 검증용 데이터프레임을 만듭니다. 이 데이터프레임은 모델을 평가하는 데 사용될 것입니다.


5. 모델 학습

선택한 딥러닝 모델을 학습 데이터를 사용하여 학습시킵니다. 모델은 입력 데이터와 실제 혼잡도 값 간의 관계를 학습하게 됩니다.


6. 모델 평가

검증용 데이터프레임을 사용하여 모델을 평가합니다. 모델이 예측한 혼잡도와 실제 혼잡도를 비교하여 모델의 성능을 평가합니다. 평가 지표로는 평균 제곱 오차(Mean Squared Error, MSE)나 평균 절대 오차(Mean Absolute Error, MAE) 등을 사용할 수 있습니다.


7. 혼잡도 예측

모델이 학습되고 검증되면, 이제 예측할 준비가 됩니다. 모델을 사용하여 각 인천의 구에서 가장 혼잡할 것으로 예상되는 역을 예측할 수 있습니다. 이 설명을 통해, 데이터 수집부터 딥러닝 모델 학습, 평가, 예측까지의 과정을 조리 있게 이해할 수 있을 것입니다. 또한, 데이터를 샘플링하여 검증용 데이터프레임을 만들고, 원본 데이터에서 해당 행을 제거하여 학습 데이터를 생성하는 부분도 설명하였습니다
